# Step 0: Hello World

**Extends**: Nothing (Genesis)

## What changed
This is the starting point of our Java journey. It contains a simple console application to verify that the Java 25 environment is correctly set up and capable of compiling and running basic code.

## How to verify
1. Open a terminal in this directory.
2. Compile the code:
   ```bash
   javac HelloWorld.java
   ```
3. Run the compiled code:
   ```bash
   java HelloWorld
   ```
4. You should see `Hello, World!` printed to the console.
